## int
1. Целое число произвольной точности.
2. Ограничено только доступной памятью.
3. Арифметические, сравнение, битовые операции.
4. Неизменяемый.
5. Нет ограничений, кроме памяти.
6. Зависит от размера числа.
7. Поддерживает длинную арифметику.
---
## float
1. Число с плавающей точкой двойной точности.
2. Примерно от 2.2e-308 до 1.8e+308.
3. Арифметические, сравнение.
4. Неизменяемый.
5. Возможна потеря точности.
6. Обычно 64 бита.
7. Использует стандарт IEEE 754.
---
## complex
1. Комплексное число.
2. Ограничено только доступной памятью.
3. Арифметические, сравнение.
4. Неизменяемый.
5. Нет ограничений, кроме памяти.
6. Зависит от размера числа.
7. Состоит из действительной и мнимой частей.
---
## str
1. Строка символов.
2. Ограничено только доступной памятью.
3. Конкатенация, срезы, поиск, форматирование.
4. Неизменяемый.
5. Нет ограничений, кроме памяти.
6. Зависит от длины строки.
7. Поддерживает Unicode.
---
## bool
1. Логический тип (True/False).
2. True или False.
3. Логические операции, сравнение.
4. Неизменяемый.
5. Нет ограничений.
6. Обычно 1 байт.
7. Подкласс int.
---
## list
1. Упорядоченная коллекция элементов.
2. Ограничено только доступной памятью.
3. Добавление, удаление, срезы, итерация.
4. Изменяемый.
5. Нет ограничений, кроме памяти.
6. Зависит от количества элементов.
7. Может содержать элементы разных типов.
---
## tuple
1. Упорядоченная неизменяемая коллекция элементов.
2. Ограничено только доступной памятью.
3. Срезы, итерация.
4. Неизменяемый.
5. Нет ограничений, кроме памяти.
6. Зависит от количества элементов.
7. Может содержать элементы разных типов.
---
## range
1. Последовательность чисел.
2. Ограничено только доступной памятью.
3. Итерация, проверка вхождения.
4. Неизменяемый.
5. Нет ограничений, кроме памяти.
6. Зависит от длины диапазона.
7. Используется для циклов.
---
## set
1. Неупорядоченная коллекция уникальных элементов.
2. Ограничено только доступной памятью.
3. Добавление, удаление, операции с множествами.
4. Изменяемый.
5. Элементы должны быть хешируемыми.
6. Зависит от количества элементов.
7. Не поддерживает индексацию.
---
## frozenset
1. Неупорядоченная неизменяемая коллекция уникальных элементов.
2. Ограничено только доступной памятью.
3. Операции с множествами.
4. Неизменяемый.
5. Элементы должны быть хешируемыми.
6. Зависит от количества элементов.
7. Не поддерживает индексацию.
---
## dict
1. Коллекция пар ключ-значение.
2. Ограничено только доступной памятью.
3. Добавление, удаление, поиск, итерация.
4. Изменяемый.
5. Ключи должны быть хешируемыми.
6. Зависит от количества элементов.
7. Неупорядоченный до Python 3.7.
---
## bytes
1. Неизменяемая последовательность байтов.
2. Ограничено только доступной памятью.
3. Срезы, итерация, преобразование.
4. Неизменяемый.
5. Нет ограничений, кроме памяти.
6. Зависит от длины последовательности.
7. Используется для бинарных данных.
---
## bytearray
1. Изменяемая последовательность байтов.
2. Ограничено только доступной памятью.
3. Срезы, итерация, добавление, удаление.
4. Изменяемый.
5. Нет ограничений, кроме памяти.
6. Зависит от длины последовательности.
7. Используется для бинарных данных.
---
## memoryview
1. Представление памяти объекта.
2. Ограничено только доступной памятью.
3. Срезы, итерация.
4. Зависит от исходного объекта.
5. Нет ограничений, кроме памяти.
6. Зависит от размера объекта.
7. Используется для работы с памятью.
---
## NoneType
1. Отсутствие значения.
2. Единственное значение None.
3. Нет операций.
4. Неизменяемый.
5. Нет ограничений.
6. Обычно 16 байт.
7. Используется для обозначения пустоты.
---
## Decimal (из модуля decimal)
1. Десятичное число с фиксированной точностью.
2. Ограничено только доступной памятью и заданной точностью.
3. Арифметические, сравнение, математические функции.
4. Неизменяемый.
5. Требует явного указания точности (количество знаков после запятой).
6. Зависит от размера числа и точности.
7. Используется для точных финансовых и денежных расчетов.
---
## Fraction (из модуля fractions)
1. Рациональное число (дробь), представленное как числитель и знаменатель.
2. Ограничено только доступной памятью.
3. Арифметические, сравнение, математические функции.
4. Неизменяемый.
5. Числитель и знаменатель должны быть целыми числами.
6. Зависит от размера числителя и знаменателя.
7. Используется для точных вычислений с дробями.
---
## array (из модуля array)
1. Однородный тип данных, компактное хранение, низкоуровневый интерфейс.
2. Зависит от типа данных (например, 'i' для целых чисел, 'f' для чисел с плавающей точкой).
3. Индексация, добавление, удаление, изменение элементов, срезы.
4. Изменяемый (mutable).
5. Ограниченный набор типов данных, менее гибкий по сравнению с list.
6. Меньше, чем list, за счет однородности данных.
7. Эффективен для числовых операций, требует меньше памяти.
---
## deque (из модуля collections)
1. Двусторонняя очередь, быстрые операции добавления/удаления с обоих концов.
2. Любые объекты Python.
3. Добавление/удаление с начала и конца, индексация, итерация.
4. Изменяемый (mutable).
5. Оптимизирован для операций с концами, но медленнее для операций в середине.
6. Больше, чем list, из-за дополнительной структуры данных.
7. Потокобезопасен для операций с концами, реализован на основе двусвязного списка.
